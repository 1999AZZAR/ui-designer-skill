# Editorial Genre

The canonical anti-slop voice. Default for most content-driven briefs.

## Character
- Typography-driven, restrained palette
- Whitespace as active layout tool
- Content hierarchy via type scale, not color

## Theme Cluster
Specimen, Atelier, Brutal, Newsprint, Studio, Manifesto, Garden, Riso, Sport, Editorial, Carnival

## Slop Gate Overrides
- Editorial tightens: gate 2 (section tags) applies at full strictness
- Editorial loosens: gate 23 (OKLCH over hex) — hex allowed for legacy compatibility

## Voice Fixtures
- CTA: specific action verbs (subscribe, read, explore), never "get started"
- Headlines: declarative, not imperative
- Metadata: muted, small, above body
