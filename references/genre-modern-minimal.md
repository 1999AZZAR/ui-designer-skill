# Modern-Minimal Genre

Stripe / Linear / ElevenLabs school. For SaaS, enterprise, developer tools.

## Character
- Geometric sans display, restrained accent
- Maximum content density with clean hierarchy
- Microinteractions as feedback, not decoration

## Theme Cluster
Coral, Cobalt

## Slop Gate Overrides
- Modern-minimal loosens: gate 23 (OKLCH over hex) — OKLCH strongly preferred but hex accepted in tokens
- Modern-minimal tightens: gate 7 (CTA voice) — CTA must be specific, never "learn more"
- Modern-minimal skips: gate 37 (glassmorphism backdrop) — does not apply

## Voice Fixtures
- CTA: verb + benefit (deploy in minutes, ship faster)
- Headlines: short, benefit-led, ≤7 words
- Labels: tool-like, functional
