# Editorial Landing Archetype

Use for magazines, journals, reports, founder letters, product stories, and typography-first branded pages.

## Best System Fits

- `swiss`
- `minimal`
- `neo-m3`
- `apple-hig` for calmer premium variants

## Core Regions

- strong masthead or opening statement
- asymmetrical hero composition
- modular story or feature blocks
- supporting navigation or section index
- concluding CTA or continuation path

## Behavior Rules

- Let typography and rhythm do the heavy lifting
- Avoid over-boxing content
- Use imagery selectively and with intention
- Treat whitespace as structure, not leftover space

## Good Hybrids

- `swiss` + `minimal` for disciplined editorial clarity
- `neo-m3` + `minimal` for stronger tech-media personality

## Starter Structure

```text
Masthead
Hero Statement
Feature Modules
Section Index or Navigation
Closing CTA
```
