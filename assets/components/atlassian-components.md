# Atlassian Components

Use these patterns for collaboration-heavy products, task management, and teamwork workflows.

## Page Scaffold

- Product header with contextual actions
- Left navigation or project sidebar
- Main task or board region
- Secondary activity, comments, or detail panel

## Component Priorities

- Primary: button, lozenge/status, modal, inline edit, board/list item, sidebar nav
- Supporting: flags, empty states, avatars, breadcrumbs, tabs

## Behavior Rules

- Make collaboration context visible
- Keep actions discoverable and direct
- Use status color consistently across board, detail, and notifications
- Optimize for shared understanding, not just individual speed

## Tailwind Notes

- 8px spacing system
- Modest radii and layered surfaces
- Primary: `#0052CC`

## Example Pattern

- Task board
  - project header
  - filters and assignee controls
  - board columns with cards
  - detail modal or side panel
