# M3 Pastel Components

Use these patterns for soft modern SaaS or creative tools with Material structure and gentler color.

## Component Priorities

- Primary: large-radius cards, tonal buttons, pastel chips, floating panels
- Supporting: soft gradients, light glass layers, friendly forms

## Behavior Rules

- Keep surfaces calm and optimistic
- Preserve Material clarity under softer visual treatment
- Avoid low-contrast pastel-on-pastel text
