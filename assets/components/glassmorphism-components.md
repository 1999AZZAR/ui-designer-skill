# Glassmorphism Components

Use these patterns for atmospheric surfaces, hero sections, and branded modern UI.

## Component Priorities

- Primary: frosted cards, translucent nav bars, floating panels, gradient buttons
- Supporting: glow accents, layered backgrounds, glass overlays

## Behavior Rules

- Keep content legible over blur and transparency
- Use glass as contrast against a richer backdrop
- Avoid stacking too many translucent layers
