# Material Components

Use these patterns for adaptive, modern product interfaces with expressive yet structured behavior.

## Component Priorities

- Primary: top app bar, navigation rail, cards, FAB, dialogs, bottom sheets, tabs
- Supporting: chips, snackbars, segmented buttons, tonal surfaces

## Behavior Rules

- Use color and elevation to separate function layers
- Keep layouts adaptive across breakpoints
- Favor large, legible touch targets and clear hierarchy
