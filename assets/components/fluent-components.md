# Fluent Components

Use these patterns for Microsoft-adjacent enterprise products and Windows-like interfaces.

## Page Scaffold

- Command bar or top action strip
- Modular content areas with layered surfaces
- Optional left rail navigation

## Component Priorities

- Primary: command bar, card, list, panel, dialog, pivot/tabs, teaching callout
- Supporting: acrylic surface, reveal interaction, badges, message bars

## Behavior Rules

- Use depth, light, and motion to clarify interaction
- Keep surfaces layered and purposeful
- Favor calm, professional visuals with subtle material effects
- Do not overuse acrylic everywhere

## Tailwind Notes

- Use blur and layered shadows carefully
- Prefer clean typography and Windows-like density
- Primary: `#0078D4`

## Example Pattern

- Enterprise workspace
  - command bar
  - card-based dashboard
  - side panel and dialog flows
