# Neumorphism Components

Use these patterns sparingly for soft, tactile interfaces where atmosphere matters more than density.

## Page Scaffold

- Large padded canvas
- Few component types
- Clear separation between raised and inset surfaces

## Component Priorities

- Primary: raised cards, pill buttons, inset inputs, toggles, knobs
- Supporting: soft icon buttons, minimal dividers, monochrome surfaces

## Behavior Rules

- Keep contrast usable even with the soft style
- Limit the number of simultaneous elevations
- Prefer monochrome or near-monochrome palettes
- Avoid using this style for dense enterprise workflows

## Tailwind Notes

- Large radii
- Dual shadow system
- Low-contrast but still accessible surfaces

## Example Pattern

- Music or wellness control panel
  - raised module cards
  - inset sliders and controls
  - minimal navigation
