# Neo-Brutalism Components

Use these patterns for highly expressive interfaces with bold structure and strong graphic energy.

## Component Priorities

- Primary: boxed cards, thick-border buttons, hard-shadow panels, oversized labels
- Supporting: sticker-like tags, punched color blocks, playful dividers

## Behavior Rules

- Keep borders and shadow logic consistent
- Use contrast and scale as the main hierarchy tools
- Maintain readability despite the aggressive style
